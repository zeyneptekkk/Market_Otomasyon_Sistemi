Ürün Yönetimi

Var olan ürünü güncelleme

Ürün silme

Kullanıcı Yönetimi

Yeni kullanıcı (kasiyer/admin) ekleme

Kullanıcı bilgilerini güncelleme

Kullanıcı silme

Depo Yönetimi

Stok takibi

Ürün listesi görüntüleme
